## Overview
This project performs comprehensive financial time series and econometric analysis on gold and related financial indices using R. It integrates statistical hypothesis testing, regression modeling, principal component analysis (PCA), seasonal decomposition, and ARIMA forecasting to evaluate market relationships, price behavior, and predictive performance.

The project demonstrates practical applications of econometrics and time series analysis using real-world financial data.

---

## Key Features

- Log-based Price Elasticity Analysis  
- Granger Causality Testing  
- Event Impact Analysis (Regression-Based)  
- Principal Component Analysis (PCA)  
- Seasonal Decomposition of Time Series  
- Multiple Linear Regression Modeling  
- ARIMA Forecasting (Auto ARIMA)  
- Model Evaluation using MAE, MSE, RMSE  
- Goodness-of-Fit Testing (Chi-Square)  
- Comparative Visualization of Actual vs Predicted Values
